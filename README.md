# 📱 College Application (Android App)

## 📌 Overview

This is an Android-based College Application developed using Android Studio.
The app provides an easy way for students and admins to manage college-related activities.

---

## 🚀 Features

* 🔐 User Login System
* 🎓 Student Dashboard
* 🛠️ Admin Dashboard
* 📢 Notifications / Updates
* 📂 Easy Navigation UI
